Account Info

Get information about the authenticated user. Requires authentication.

  • Command

    GET /account/info

  • Sample Request

    GET https://api.badger.com/account/info?auth_code=ABC123

  • Sample Response

    { "domain_credits": 42, "name": "John Smith", "invites_available" : 5, "confirmed_email" : true }

Request Invite

Send a Badger invitation request for the provided email address.

  • Command

    POST /account/request_invite

  • Parameters (required values are in bold)

    Name Description
    email Email address for person requesting invite

  • Sample Request

    POST https://api.badger.com/account/invite_request?email=johnsmith@test.com

  • Sample Response

    { "message": "Ok", "invite_request_id": "1-johnsmith@test.com" }

Update Invite Request Extra Information

Update full name, total domains registered, and suggestions of invite request.

  • Command

    POST /account/request_invite_extra_info

  • Parameters (required values are in bold)

    Name Description
    invite_request_id The invite request ID
    full_name The full name of requester
    total_domains_registered The number of domains requester has registered
    suggestions The suggestions for badger.com

  • Sample Request

    POST https://api.badger.com/account/request_invite_extra_info?invite_request_id=1-johnsmith@test.com&full_name=johnsmith&total_domains_registered=11-50&suggestion=suggestion1

  • Sample Response

    { "message": "Ok" }

Create Account

Create a new account with Badger.

  • Command

    POST /account/

  • Parameters (required values are in bold)

    Name Description
    invite_code Invitation code sent by Badger.
    first_name First name for account owner
    last_name Last name for account owner
    email Email address for account owner
    password Password of account owner
    agree_to_terms User have to agree to terms of service of Badger
    country The account owners specified country
    locale The account owners specified locale
    gender The account owners specified gender

  • Sample Request

    POST https://api.badger.com/account?first_name=John&last_name=Smith&email=johnsmith@test.com&password=badger42!&invite_code=abc123&agree_to_terms=true

  • Sample Response

    { "message": "Account created", "access_token": "11.11111111.111111aaaabbbbcccccdddd000" }

Reset Password

Given a reset code (sent in email) and correct email address, reset an account password.

  • Command

    POST /account/reset_password

  • Parameters (required values are in bold)

    Name Description
    code The password reset code sent by email.
    email The email address for the account.
    new_password The new password for the account.

  • Sample Request

    POST https://api.badger.com/account/reset_password?email=johnsmith@test.com&code=abc123&new_password=badger42?

  • Sample Response

    { "message": "Password reset" }

Send Invitation

Invite friend to use badger.com. Requires authentication

  • Command

    POST /account/send_invite

  • Parameters (required values are in bold)

    Name Description
    email The email of invited friend.
    first_name The first name of invited friend.
    last_name The last name of invited friend.
    credits_to_gift The amount of credits to transfer to friend.
    custom_message Custom message to friend.

  • Sample Request

    POST https://api.badger.com/account/send_invite?email=myemail@example.com&first_name=FirstName&last_name=LastName&credits_to_gift=3&custom_message=message

  • Sample Response

    { "message": "An invitation for badger.com has been sent to myemail@example.com" }
